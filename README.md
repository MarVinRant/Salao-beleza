# 💇‍♀️ Salão de Beleza & Estética - Landing Page

## 📖 Sobre o Projeto
Uma landing page sofisticada, moderna e visualmente impactante, desenvolvida para salões de beleza, estúdios de maquiagem e profissionais da estética. O objetivo principal deste projeto é transmitir requinte e profissionalismo, criando uma vitrine digital que valoriza o trabalho do profissional e facilita o agendamento de horários pelos clientes.

Este projeto compõe meu portfólio de desenvolvimento Front-End, destacando minha capacidade de criar interfaces elegantes e funcionais para o setor de beleza e cuidados pessoais.

## ✨ Principais Funcionalidades
* **Design Elegante e Sofisticado:** Paleta de cores e tipografia escolhidas a dedo para refletir a estética premium que o setor de beleza exige.
* **Mobile-First & Responsividade:** Interface perfeitamente adaptada para smartphones, garantindo que as clientes consigam ver os serviços e agendar horários de qualquer lugar, na palma da mão.
* **Exposição de Serviços:** Seções claras e objetivas para apresentar cortes, colorações, maquiagens e outros tratamentos, gerando desejo através do visual.
* **Agendamento Facilitado (CTA):** Botões de chamada para ação estrategicamente posicionados para direcionar o cliente rapidamente ao WhatsApp ou sistema de reservas.
* **Alta Performance:** Animações suaves e carregamento rápido de imagens, garantindo uma experiência de navegação fluida e agradável.

---

## 🛠️ Tecnologias Utilizadas
A página foi construída priorizando um código limpo, controle absoluto do layout e máxima performance, sem a necessidade de frameworks de terceiros:

* **HTML5:** Estruturação semântica para garantir boa indexação em mecanismos de busca (SEO local).
* **CSS3:** Estilizações modernas, efeitos de transição suaves, Flexbox/Grid Layout e Media Queries para adaptação de telas.
* **JavaScript (Vanilla):** Implementação de interatividade e comportamentos dinâmicos na interface.
